# Padrões de Projeto em Python — Singleton, Adapter e Strategy

Este repositório contém a implementação de três padrões de projeto fundamentais da Engenharia de Software: **Singleton (Criacional)**, **Adapter (Estrutural)** e **Strategy (Comportamental)**.  
O objetivo é demonstrar, de forma simples e prática, como cada padrão funciona, qual problema resolve e como pode ser aplicado em situações reais.

As implementações são baseadas nos conceitos apresentados pelo site **Refactoring Guru**: https://refactoring.guru/pt-br/design-patterns/catalog.

Cada pasta contém:
- Código de exemplo do padrão
- Explicação detalhada em README próprio
- Diagrama UML ilustrando a estrutura

O projeto foi desenvolvido utilizando **Python** no VSCode, com apoio da LLM **ChatGPT** para organização e explicação do código.
